# Tailwind CSS Project Clone 🚀

A simple and responsive UI clone built using **Tailwind CSS**. This project demonstrates how to use Tailwind’s utility-first approach to design modern web interfaces efficiently.

---

## 📂 Project Structure

```
Tailwindcss-Project-Clone/
│── index.html        # Main HTML file
│── input.css         # Tailwind source file
│── output.css        # Compiled Tailwind CSS
│── image/            # Images and assets
│── README.md
```

---

## 📌 Overview

This project is a frontend clone created for practicing **Tailwind CSS**. It focuses on building responsive layouts and styling components using Tailwind utilities instead of writing custom CSS.

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ayush87914/Tailwindcss-Project-Clone.git
```

### 2. Navigate into the folder

```bash
cd Tailwindcss-Project-Clone
```

### 3. Run the project

Open `index.html` in your browser.

---

## ⚙️ Tailwind Setup (Optional)

If you want to rebuild the CSS:

1. Install Tailwind CSS
2. Run the build command:

```bash
npx tailwindcss -i ./input.css -o ./output.css --watch
```
